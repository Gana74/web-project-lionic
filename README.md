# Lionic

## Описание

Lionic — это проект, включающий HTML, CSS и JavaScript, с использованием различных библиотек и шрифтов. Основное внимание уделяется адаптивному дизайну и мультимедийному контенту.

## Используемые технологии

- HTML5
- CSS3
- JavaScript
- Bootstrap
